**Pejuang Ngabuburit** adalah game *turn-based* RPG bertema lokal yang dibangun dengan estetika 16-bit pixel art. Game ini menantang pemain untuk bertahan dari godaan berbagai "Monster Takjil" hingga waktu berbuka puasa tiba.

## 🎮 Mekanik Game
* **Iman (HP):** Indikator ketahananmu. Jika mencapai 0, kamu dinyatakan "mokel" (Game Over).
* **Fokus (MP):** Dibutuhkan untuk menggunakan skill *Sabar* atau *Fokus*.
* **Time Progression:** Setiap giliran menambah waktu 5 menit. Musuh menjadi 25% lebih kuat setiap jamnya.
* **Dynamic Background:** Langit berubah warna secara real-time dari siang ke *golden hour* seiring mendekatnya waktu Maghrib.

## 👾 Bestiary (Daftar Musuh)
* **Gorengan Squad:** Musuh tipe api yang berminyak dan panas.
* **Es Buah Elemental:** Musuh tipe es yang bisa memberikan efek *brain freeze*.
* **Iklan Sirop Legend:** Musuh tangguh yang muncul saat mendekati waktu berbuka.
* **Great Kolak Naga (Final Boss):** Naga legendaris yang terbuat dari gula jawa dan santan.

## 🎒 Inventory & Items
* **Kurma:** Memulihkan Iman (HP) dalam jumlah besar.
* **Air Putih:** Memulihkan Fokus (MP) untuk mengeluarkan skill.
* **Minyak Kayu Putih:** Memberikan *Defense Buff* agar serangan musuh tidak terlalu terasa.

## 🛠️ Tech Stack & Pengembangan
* **Core:** HTML5 Canvas & Vanilla JavaScript.
* **Visual Engine:** Custom Pixel Matrix Rendering (Rendering manual via array warna).
* **Typography:** 'Press Start 2P' via Google Fonts.
* **Status Proyek:** Tugas pengembangan mandiri (Semester 6 - Universitas Bina Sarana Informatika).
